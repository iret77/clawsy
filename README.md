To build Clawsy from source:

1.  **Prerequisites:** Xcode (or Command Line Tools) installed.
2.  **Clone:** `git clone https://github.com/openclaw/clawsy.git`
3.  **Build:** Run `./build.sh`
    *   This compiles the Swift code (Release mode).
    *   Packages it into `Clawsy.app` inside `.build/app/`.
4.  **Run:** Open `.build/app/Clawsy.app`
