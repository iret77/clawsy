# Clawsy: The Mac Bridge for OpenClaw

Clawsy is the essential companion for any agent that needs to interact with a user's physical macOS environment. It provides a secure, sandboxed bridge for seamless workflows.

## Capabilities
- **File Sync**: Automatically sync files between the agent's workspace and the user's Mac.
- **Visuals**: Trigger high-res screenshots or live camera snapshots.
- **Clipboard**: Read from or write to the macOS clipboard.
- **Dashboard**: Report task progress directly to the user's desktop Mission Control.

## How to use
Tell Clawsy to perform actions on the Mac. For example:
- "Take a screenshot and analyze the UI."
- "Move the generated PDF to the user's Clawsy folder."
- "Sync the latest project files from the Mac."

## Installation
The user must have the Clawsy app installed and running on their Mac.
